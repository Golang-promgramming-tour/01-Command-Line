# tour 工具集

tour 工具集是《Go 语言编程之旅：一起用 Go 做项目》中的项目，是第一章 [命令行应用：打造属于自己的工具集] 的附属源码。

## 目录

- 第1章 命令行应用：打造属于自己的工具集
    - 1．1 工具之旅
    - 1．2 单词格式转换
    - 1．3 便捷的时间工具
    - 1．4 SQL语句到结构体的转换

## 功能清单

- 单词格式转换
- 便利的时间工具
- SQL语句到结构体的转换 
- JSON 到结构体的转换
